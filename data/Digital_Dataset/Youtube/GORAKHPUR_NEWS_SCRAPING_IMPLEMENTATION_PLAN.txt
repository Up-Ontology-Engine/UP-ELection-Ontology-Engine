================================================================================
Project: Automated News & Comment Scraping and Political Sentiment Analysis
Date: May 2026
Objective: Comprehensive collection and classification of Gorakhpur-related news 
and YouTube comments into pro-BJP and anti-BJP categories using AI

================================================================================
1. PROJECT OVERVIEW
================================================================================

This implementation plan outlines a multi-stage pipeline to:
  a) Scrape all news articles mentioning Gorakhpur from various sources
  b) Store news data as JSON files in structured directories
  c) Automatically classify news into pro-BJP and anti-BJP categories using NLP
  d) Scrape YouTube videos related to Gorakhpur
  e) Extract comments from these videos
  f) Classify comments into pro-BJP and anti-BJP sentiment categories
  g) Store all data in organized JSON formats for analysis

Target Platforms:
  - News Sources: Google News, News APIs (NewsAPI, India Today, Jagran, etc.)
  - Video Platforms: YouTube
  - Comment Data: YouTube comments API
  - Regional Content: Hindi/English news sources specific to Gorakhpur

3. CLASSIFICATION CATEGORIES
================================================================================

PRIMARY CLASSIFICATION: Pro-BJP vs Anti-BJP

Pro-BJP Indicators:
  - Positive mentions of BJP policies and governance
  - Support for current government initiatives
  - Criticism of opposition parties
  - Praise for BJP leadership and ministers
  - Positive economic/development reports attributed to BJP
  - Keywords: "development", "growth", "stability", "security"

Anti-BJP Indicators:
  - Criticism of BJP policies or governance
  - Support for opposition parties
  - Negative reports on government initiatives
  - Concerns about governance or development
  - Keywords: "corruption", "mismanagement", "failure", "injustice"

NEUTRAL CLASSIFICATION:
  - Factual reporting without sentiment
  - Equal coverage of multiple parties
  - Non-political Gorakhpur news

CONFIDENCE SCORE: 0.0-1.0 (AI confidence in classification)

6. CLASSIFICATION ALGORITHM DETAILS
================================================================================

Approach 1: Zero-Shot Classification (Fastest)
──────────────────────────────────────────────
Model: facebook/bart-large-mnli
Process:
  1. Input text (article or comment)
  2. Define candidate labels: ["pro-BJP", "anti-BJP", "neutral"]
  3. Model outputs probability for each label
  4. Threshold-based decision (default: 0.5)
  5. Return highest probability label with confidence

Advantages: Fast, no fine-tuning, multilingual support
Disadvantages: May miss nuanced context

Approach 2: Sentiment Analysis + Keyword Matching (Balanced)
────────────────────────────────────────────────────────────
Process:
  1. Extract sentiment using transformers (positive/negative/neutral)
  2. Entity recognition for political figures/organizations
  3. Keyword matching against pro-BJP and anti-BJP lexicons
  4. Combine signals with weighted scoring:
     score = 0.4*sentiment + 0.4*keywords + 0.2*entities
  5. Classify based on aggregated score

Advantages: Balanced accuracy, interpretable
Disadvantages: Requires keyword lexicon maintenance

Approach 3: Fine-tuned Model (Best Accuracy)
─────────────────────────────────────────────
Process:
  1. Collect training data (1000-5000 labeled examples)
  2. Fine-tune DistilBERT on labeled dataset
  3. Use fine-tuned model for inference
  4. Regular re-training with new data

Advantages: High accuracy, context-aware
Disadvantages: Requires labeled training data, longer inference time

RECOMMENDED: Ensemble approach combining all three

Language Support:
  - English: BERT, DistilBERT, RoBERTa
  - Hindi: Bhashini API, IndicBERT
  - Language detection: langdetect or textblob
  - Translation: Google Translate API (fallback)

Parameters:
